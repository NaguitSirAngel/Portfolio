Web Development Knowledge and Tools

  • HyperText Markup Language

    Standard Markup Language that is displayed on a web page. Can be assisted by CSS to improve appearance.

  • Cascading Style Sheets

    Used with html to improve a web page's visual presentation. It can manipulate web page layouts all at once.

  • Bootstrap

    A free open-source CSS framework that will help a web page to be mobile responsive. It also provides pre-made elements such as buttons, forms and more.

  • JavaScript

    Used mainly for Web Development. Along with HTML and CSS, JavaScript is one of the main technologies of the World Wide Web.

  • Node.js

    An open-source JavaScript runtime environment. Mostly used for SPA's (Single Page Applications).

  • Angular

    A typescript based web application development framework by Google. Manipulating elements on a webpage is made easier by the use of its amazing directives.

  • Typescript

    Developed by Microsoft, typescript is an open-source programming language. It is used along with the Angular framework since the Angular was built entirely with Typescript.

  • Express

    A web framework that is used and designed for developing APIs. It is the de facto standard server framework in Node.js. It is an open-source software under MIT License.

  • ASP .NET

    An open-source server side web-application framework developed by Microsoft. Enables developers to create dynamic web pages. It uses the C# language.

  • Java Web Development

    The Java programming language enables developers to create dynamic web applications along with the use of powerful IDE's such as Eclipse and Netbeans.
